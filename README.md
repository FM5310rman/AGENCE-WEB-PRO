# WebPro - Agence Web Professionnelle

## 📋 Description

WebPro est un site web complet et professionnel pour une agence de création web. Le projet est conçu en HTML5 sémantique, CSS3 moderne avec Flexbox et Grid, et JavaScript vanilla pour l'interactivité.

## 🎯 Objectifs du Projet

- ✅ Concevoir un site web complet de A à Z
- ✅ Appliquer HTML5 sémantique professionnel
- ✅ Utiliser Flexbox & Grid pour des layouts complexes
- ✅ Rendre un site 100% responsive
- ✅ Ajouter des animations et effets modernes
- ✅ Livrer un projet prêt pour un client

## 🚀 Comment Utiliser

### Installation
1. Cloner ou télécharger le projet
2. Ouvrir `index.html` dans un navigateur
3. Aucune dépendance externe requise!

### Personnalisation
1. Modifier les couleurs dans `css/style.css` (variables CSS)
2. Remplacer les images dans le dossier `images/`
3. Mettre à jour le contenu HTML
4. Adapter le JavaScript selon les besoins
